# Сказка о рыбаке и рыбке — UML

Проект содержит UML-диаграммы, описывающие систему обработки запросов.

 Диаграммы
- Use Case — сценарии взаимодействия
- Class — структура классов
- Activity — процесс выполнения запросов
- Sequence — последовательность взаимодействий
- State — состояния системы (Старухи)

 Идея
Система моделирует обработку запросов с ограничением ресурсов.
При превышении лимита происходит сброс к начальному состоянию.
